### Quantize(量化)
domain连续，range散点
d3.scaleQuantize()
invertExtent(value)
```
var width = d3.scaleQuantize()
    .domain([10, 100])
    .range([1, 2, 4]);
width.invertExtent(2); // [40, 70]
```

### Threshold(阈值)比例尺
d3.scaleThreshold()
```
var color = d3.scaleThreshold()
    .domain([0, 1])
    .range(["red", "white", "green"]);

color(-1);   // "red"
color(0);    // "white"
color(0.5);  // "white"
color(1);    // "green"
color(1000); // "green"
```
